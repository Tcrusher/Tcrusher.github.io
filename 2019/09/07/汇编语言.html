汇编语言

发表于 2019-09-07 | 更新于 2019-09-21 | 分类于 汇编语言 | 评论数： | 阅读次数：

机器语言编写程序绝对是让人一件头疼的事情，为了避免使用0和1的组合来编写程序。汇编语言使用短语（缩写）来表示指令，使用变量名称来表示存储空间的位置。汇编语言指令与机器语言指令之间存在着一对一的关系。

汇编语言的学习，一点笔记。。。

1、汇编语言介绍

汇编语言（assembly language）是一种用于电子计算机、微处理器、微控制器或其他可编程器件的低级语言，亦称为符号语言。在汇编语言中，用助记符代替机器指令的操作码，用地址符号或标号代替指令或操作数的地址。在不同的设备中，汇编语言对应着不同的机器语言指令集，通过汇编过程转换成机器指令。特定的汇编语言和特定的机器语言指令集是一一对应的，不同平台之间不可直接移植

—-来自百度百科

2.2、变量声明

变量名必须以字母打头，其后可以跟着字母数字，可以包含‘_’，’@’，’$’特殊符号，但是通常避免使用特殊符号。变量名不区分大小写，最大长度为247个字符，通常变量名有1~10个字符组成。

类型 占比特位数 表示范围（包含）
sdword 32
dword 32
sword 16
word 16
sbyte 8 -128~127
byte 8 0~255

2.3、立即数

mov指令将右边操作数值指派给左边的操作数，右边的操作数称为源操作数，左边的操作数称为目的操作数。

指令 意义
mov mem, imm 将立即数存储到内存中
mov reg, imm 将内存单元的值存储到寄存器中
mov mem, reg 将寄存器中的内容存储到内存单元中
mov reg, imm 将立即数存储到寄存器中
mov reg, reg 将源（第二）寄存器中的内容存储到目的（第一）寄存器中

imm：immediate

mem：memory

reg：register

2.4、寄存器

数据不能直接从一个内存单元移动到另一个内存单元中。

先把随机数存储器（RAM）当中某一个存储单元的内容复制或装载到中央处理器（CPU）中，然后再从中央处理器中把这个内容复制或存储回随机存储器的某一个单元中。在CPU中实现上述快速地、临时性的存储空间作用的部件称为存储器，也称累加器

2.5、数据移动

如果实现num2=num1；因为不能从一个内存单元移动数据到另一个内存单元，因此不能写成mov num2, num1；必须需要把num1的内容复制到寄存器中，然后将寄存器中的内容复制到num2中

4、算术运算指令

4.1、加法与减法运算

  • 加法指令
指令 意义
add mem, imm 将内存单元的数值与立即数的值相加，结果值放到内存单元中
add reg, mem 将寄存器中的值加上内存单元中的值，结果值放到寄存器中
add mem, reg 将内存单元中的值加上寄存器中的值，结果值放到内存单元中
add reg, imm 将寄存器中的值加上立即数的值，结果值放到寄存器中
add reg, reg 将目的(第一个)寄存器的值加上源(第二个)寄存器的值，结果值放到目的寄存器中
  • 减法指令
指令 意义
sub mem, imm 将内存单元的数值减去立即数的值，结果值放到内存单元中
sub reg, mem 将寄存器中的值减去内存单元中的值，结果值放到寄存器中
sub mem, reg 将内存单元中的值减去寄存器中的值，结果值放到内存单元中
sub reg, imm 将寄存器中的值减去立即数的值，结果值放到寄存器中
sub reg, reg 将目的(第一个)寄存器的值减去源(第二个)寄存器的值，结果值放到目的寄存器中

4.2、乘法与除法运算

  • 乘法指令
指令 意义
imul reg 将寄存器eax的值与寄存器中的整数值相乘
imul mem 将内存单元中的整数值与寄存器eax中的值相乘
  • idiv指令-除法指令
指令 意义
idiv reg 将寄存器中的值拆分开存储带寄存器对edx:eax中
idiv mem 将内存中的值拆分开存储到寄存器对edx:eax中
  • 转化指令
指令 意义 描述
cbw 字节转化为字 寄存器al的内容扩展到寄存器ax
cwb 字转化为双字 寄存器ax的内容扩展到寄存器eax
cbq 双字转化为4个字 寄存器eax的内容扩展到寄存器对edx:eax

6.3、固定迭代循环for

.repeat和.untilcxz汇编指令实现

.repeat和.untilcxz指令使用ecx寄存器作为计数器，.untilcxz完成两个任务，把寄存器ecx的值减1，然后当寄存器ecx的值不为0时，它将跳转到.repeat指令处执行，也就是：当寄存器ecx的值为0时（cxz），循环停止执行。for循环是前置循环，而.repeat和.untilcxz指令是后置循环

说明：

首先，将需要执行的次数装载到寄存器ecx中，然后。untilcxz执行一次，寄存器ecx的值减1，同时与0比较，如果不为0，继续循环，为0，则程序控制流跳转到.untilcxz之后开始执行
